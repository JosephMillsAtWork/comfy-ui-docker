# ComfyUI Docker - Legacy CUDA Branch

This branch is optimized for NVIDIA RTX 30-series (Ampere) and older GPUs.

Hardware Supported:
- RTX 30-series (3060, 3070, 3080, 3090)
- RTX 20-series (Turing)
- GTX 10-series (Pascal)

Tech Stack:
- Base: PyTorch 2.5.1
- CUDA: 12.4 (Stable)

Usage:
1. git checkout legacy-cuda
2. ./start.sh
